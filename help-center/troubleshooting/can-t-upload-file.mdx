---
title: Can't Upload File
description: Solve issues uploading files to Rock.
---

## Issue

- File upload fails or gets stuck.
- Error: "Upload failed. Try again."

## Quick Fix

<Steps>
  <Step title="Check file size">
    Rock supports uploads up to 100MB per file. Large files may fail.
  </Step>
  <Step title="Refresh page">
    Sometimes a browser refresh resolves the issue.
  </Step>
  <Step title="Try another browser">
    Chrome, Firefox, and Edge are supported.
  </Step>
  <Step title="Contact support">
    If it still fails, email <code>support@rock.io</code> with a screenshot.
  </Step>
</Steps>
