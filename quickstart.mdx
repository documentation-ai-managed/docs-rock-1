## Prerequisites

<Callout kind="info" title="Before you begin">
Ensure you have:
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An email address for account verification
- Optional: Existing documentation files in Markdown, PDF, or GitHub repo for import
</Callout>

Rock provides a seamless web interface for managing your documentation. No downloads required.

## Create Your Rock Account

Follow these steps to register and log in.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://app.rock.co.uk` and click **Sign Up**.

    Fill in your email, name, and password. Accept the terms and click **Create Account**.

    Verify your email via the confirmation link sent to your inbox.
  </Step>

  <Step title="Log In" icon="log-in">
    Return to the login page and enter your credentials.

    Enable two-factor authentication (2FA) when prompted for added security.
  </Step>
</Steps>

## Set Up Your First Workspace

Workspaces organize your project documentation.

<Steps>
  <Step title="Create Workspace" icon="folder-plus">
    From the dashboard, click **New Workspace**.

    Enter a name like `My Project Docs` and select a description.

    Choose visibility: `Private` (default) or `Public`.
  </Step>

  <Step title="Customize Settings" icon="settings">
    Set your brand color to `#ED1C24` in workspace settings.

    Add team members via email invites if collaborating.
  </Step>
</Steps>

## Import Initial Content

Rock supports multiple import methods.

<Tabs>
  <Tab title="Upload Files" icon="upload">
    Drag and drop Markdown (`.md`), PDF, or image files into the workspace.

    Or click **Import** > **Upload** and select files from your device.

    Files appear as documents ready for editing.
  </Tab>

  <Tab title="GitHub Repo" icon="github">
    Connect your GitHub account via **Import** > **GitHub**.

    Enter repo URL like `https://github.com/yourusername/project-docs`.

    Select branch (e.g., `main`) and import structure.
  </Tab>

  <Tab title="Markdown Paste" icon="file-text">
    Create a new document and paste Markdown content directly.

    Use the editor's preview mode to check formatting.
  </Tab>
</Tabs>

## Create Your First Document

Build a sample page.

<CodeGroup tabs="Markdown,API">
  ```markdown
  # Welcome to My Docs

  ## Overview

  This is your first document in Rock.

  - Feature-rich editing
  - Collaboration tools
  - Custom themes
  ```

  ```javascript
  // Optional: Use Rock API to create via script
  const response = await fetch('https://api.example.com/workspaces/{workspaceId}/documents', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'Welcome',
      content: '# Hello Rock\nYour content here.'
    })
  });
  ```
</CodeGroup>

</Callout>
