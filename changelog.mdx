## Overview

Stay informed about the latest releases, bug fixes, and improvements to Rock. You receive notifications for major updates through your dashboard at `https://dashboard.example.com`.

<Callout kind="info">
Rock follows semantic versioning (SemVer): `MAJOR.MINOR.PATCH`. 
- **MAJOR**: Breaking changes
- **MINOR**: New features backward-compatible
- **PATCH**: Bug fixes backward-compatible
</Callout>

| Version Type | Impact | Example |
|--------------|--------|---------|
| `MAJOR` (e.g., 2.0.0) | Breaking changes | API restructuring |
| `MINOR` (e.g., 1.1.0) | New features | Added search functionality |
| `PATCH` (e.g., 1.0.1) | Bug fixes | Fixed rendering issues |

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated Mermaid diagram support (code block only)
- Added dark mode toggle in the documentation viewer

## Bug Fixes

- Corrected JSX attribute parsing for arrays like `tags={["feature"]}`
- Fixed table rendering on narrow screens

## Improvements

- Optimized image loading with lazy attributes

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## Breaking Changes

- Removed legacy `<Mermaid>` component; use code blocks instead
- Updated attribute syntax to require double quotes: `title="Text"`

## New Features

- Full MDX component suite: Steps, Tabs, Cards, Update, and more
- API documentation components: `<ParamField>`, `<Request>`, `<Response>`

## Initial Setup

- Core documentation rendering engine
- Frontmatter validation and parsing

</Update>
